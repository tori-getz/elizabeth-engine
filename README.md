## Dev

firstly install <b> SGDK </b> and <b>NodeJS</b>.

<b> Run this commands: </b>
```
> npm install
> git clone https://github.com/tori-getz/elizabeth-engine
> cd elizabeth-engine
> npm install
> npm run electron-dev
```
